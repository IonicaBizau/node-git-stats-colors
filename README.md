> Adds colors to the git-stats inputs.

## Installation

```sh
$ npm i --save git-stats-colors
```

## Documentation

### `GitStatsColors(input, thm)`
Adds colors to the git-stats inputs.

#### Params
- **String** `input`: The input string.
- **String|Object** `thm`: The theme object or name.

#### Return
- **String** The colored result.
